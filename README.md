twitchbook-chrome
=================

Chrome Extension to bring Twitch emoticons to Facebook. It can be found in the 
[Chrome Web Store](https://chrome.google.com/webstore/detail/ndbjnfkddkejoeekdaledhiinmppmpnj/publish-accepted
).
